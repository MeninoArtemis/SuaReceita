Instruções do Jogo

Para jogar DOOM, siga as instruções abaixo:

  • W - Andar para frente
  • A - Andar para a esquerda
  • S - Andar para trás
  • D - Andar para a direita
  • Espaço - Atirar
  • Esc - Pausar o jogo
